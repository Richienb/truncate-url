# truncate-url

> Truncate a URL to a specific length


## Install

```
$ npm install --save truncate-url
```


## Usage

```js
const truncateUrl = require('truncate-url');

truncateUrl('http://sindresorhus.com/foo/bar/baz/faz', 30);
//=> 'http://sindresorhus.com/…/faz'
```
